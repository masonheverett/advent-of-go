# Advent of Go

Learning [Golang](https://go.dev/) (v1.18) with [Advent of Code](https://adventofcode.com/)

## Running a Solution

1. Run `go install` to install the package
1. Assuming your `go/bin` is in your path, run `advent-of-go` to start the program

Usage can be found with the `advent-of-go -h` or `advent-of-go -help` command
